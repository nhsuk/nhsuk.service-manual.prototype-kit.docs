Using GitHub if you work for NHS England

Code repositories for NHS England digital teams are usually stored in the NHS Digital GitHub organisation. This is a shared account where administrators can manage all repositories. Get added to it to:

  • add a new repository that other people in the organisation can find and make changes to
  • make changes to an existing prototype
Information:

Some NHS prototype repositories are in different GitHub organisations. These repositories are being merged - do not add new repositories to them unless your team tells you to do it.

How to get added to the organisation

An NHS England GitHub admin must add you to the NHS Digital organisation - you cannot do it yourself. As your team for details on who to contact.

If you already have a GitHub account you can use the same account.

Moving a repository into the organisation

If you create a repository somewhere else by mistake (for example, on your own account), you can transfer it into the NHS Digital organisation. Ask your team for help on what to do.
