  1. Install Git.
  2. Create a GitHub account.

1. Install Git

The first thing you need to do is install Git (opens in new tab).

You do not need to set your name and email address now - you will do that later.

2. Create a GitHub account

Go to GitHub.com and create a free GitHub account.

If you work for NHS England

Code repositories for NHS England digital teams are usually stored in the NHS Digital GitHub organisation shared account.

If you need to work on a prototype stored in this organisation or add a new prototype, it is usually easier to get your account added to the organisation first. An NHS England GitHub administrator must add you - you cannot do it yourself. Ask your team for details about who to talk to.

3. Download and set up the GitHub Desktop app

Download the GitHub Desktop app.

Sign in with your GitHub account details.

You can now use GitHub Desktop to work with repositories. A repository contains all of the files in a prototype. You can use the repository to view the file history and work on your prototype with others.
