The kit stores data from all answers that users give in a prototype. This lets you use or show the answers later.

Key concepts

If you do not know how data works in the kit:

  • go to storing data in session
  • passing data from page to page
  • using routes

Using data

There are different ways of using data to change your prototype:

  • change the content within a page (called 'conditional' content)
  • send the user to different pages (called 'branching') - this means that the user will only see the pages that are relevant to them. There are different ways to do branching depending on if you are using radios or checkboxes
