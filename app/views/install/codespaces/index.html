Use the prototype kit with Github Codespaces

What is a Codespace?

A GitHub Codespace is a virtual computer in the cloud where you can write and test code (such as an NHS Prototype Kit) without needing to install anything on your own computer.

Before you begin

You do not need any technical knowledge to follow along.

You will need a Github account.

This guide will show you how to get setup in 2 key steps.

Installation typically takes up to 5 minutes.
