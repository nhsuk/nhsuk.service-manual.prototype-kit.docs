Installation on an NHS England Windows laptop

If you have a Windows laptop issued by NHS England corporate IT, then you can largely follow the install guide for Windows but there are some additional steps.

Install software dependencies

The following can be installed from the Company Portal application:

  • Terminal
  • Visual Studio Code
  • Node.js and NPM
  • Git for Windows
  • Github Desktop

Creating a local folder

Within the root of your OneDrive create a new folder named DevWspace.

This folder name was added to an approved list to allow the creation of virtual environments and allow scripts to be executed.

The path should look like this after you have created it:

"C:\Users\%username%\OneDrive - NHS\DevWspace"

Change the default path location for all the developer programs you use and move any existing prototypes you currently have to this new location.
