Before you start you will need to:

  1. create a local folder in the right place for the code to work
  2. install any software dependencies

Creating a local folder for your prototypes

Within the root of your OneDrive create a new folder named DevWspace.

The path should look like this after you have created it:

"C:\Users\%username%\OneDrive - NHS\DevWspace"

You will need to use this for all prototypes and as the path for any programmes you use for the prototype.

Install software dependencies

Go to Company Portal applications and install these appplications:

  • Terminal
  • Visual Studio Code
  • Node.js and NPM
  • Git for Windows
  • Github Desktop
