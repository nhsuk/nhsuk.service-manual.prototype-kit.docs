Services that you can use include:

- [GitHub Codespaces](https://github.com/features/codespaces) – free for around 60 hours per month
- [Gitpod](https://www.gitpod.io/) – free for around 50 hours per month

For this guide we are using GitHub Codespaces.

## Before you begin

You do not need any technical knowledge to use this guide.

You will need a [GitHub account](https://github.com/signup?ref_cta=Sign+up&ref_loc=header+logged+out).

We will show you how to:

- set up the Codespace
- work with the Codespace

Installation usually takes up to 5 minutes.
